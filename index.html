SETUP GUIDE · ~20 MIN · ONE-TIME

📈 LendPeak — Website Conversion Tracking

This lets Google Ads see which website leads turn into real customers, so it can find more of them. It's two small snippets for the web team, and almost nothing for the CRM team. Plain and simple below.

How it works in one line: the page tells Google "a lead came in" (using the person's email/phone as a private, hashed match key) → later, when that lead funds, LendPeak reports the sale back using the same email → Google learns which clicks make money. No complex plumbing — email is the link.

For the web team — 2 snippets ~15 min

1 · Add the Google tag to every landing page

Paste this in the <head> of every page under /lp/. (LendPeak will send you the real AW-XXXXXXXXX ID to drop in — it appears twice.)

<!-- Google tag (gtag.js) -->
<script async src="https://www.googletagmanager.com/gtag/js?id=AW-XXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'AW-XXXXXXXXX');
</script>

2 · Fire the "Lead" event when the form is submitted

Right after a lead is saved, call reportLead() with the email + phone the person typed. That email/phone is what lets Google match the lead later — it's hashed in the browser, so no raw personal data is sent. (LendPeak sends you the send_to value.)

<script>
function reportLead(email, phone){
  if (typeof gtag !== 'function') return;
  // Enhanced Conversions — Google hashes these automatically
  gtag('set', 'user_data', { email: email || '', phone_number: phone || '' });
  gtag('event', 'conversion', {
    send_to: 'AW-XXXXXXXXX/XXXXXXXXXXXXXXXX',   // LendPeak will send this
    value: 1.0,
    currency: 'USD'
  });
}
// Example to call on a successful submit:
// reportLead('jane@hershop.com', '+13105551234');
</script>

Phone format: send it as +1XXXXXXXXXX (country code, no spaces/dashes) when you can — it improves matching.

Optional bonus (only if quick): also save the click ID as a backup match key — add a hidden field and this one-liner to the form. Not required; email is the main link. 
<input type="hidden" name="gclid" id="gclid">
<script>var g=new URLSearchParams(location.search).get('gclid');if(g){document.getElementById('gclid').value=g;}</script>

For the CRM team (GoHighLevel) — almost nothing ~2 min check

Just confirm every lead record saves these (all standard fields you already collect — no new automation to build):
  • Email and Phone of the lead
  • The funding amount requested
  • A clear status/stage that shows when a deal is Funded
That's it — LendPeak handles reporting the funded deals back to Google from there. (If LendPeak later requests a read-only API key to pull funded deals automatically, that would be the only extra ask — but it's optional.)

✅ Quick test (5 min)

  1. Open a landing page and submit one test lead (use a fake phone like (424) 555-0142 so no real line rings).
  2. In Google's Tag Assistant, confirm the Lead event fired on submit.
  3. Confirm the test lead appears in GoHighLevel with its email + phone.
  4. Tell LendPeak it's done — we delete the test and switch the campaign on.
