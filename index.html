SETUP · ~15 MIN · ONE-TIME

📈 Install the LendPeak Google Ad Tag

Put our Google Ads tag on the 5 landing pages so leads are recorded and the campaign can optimize for more of them. Three short snippets, same on all 5 pages. This is the only setup we need from your team.

Our tag ID: AW-18139071660. Everything below uses it. Apply all three snippets to each of these 5 pages:
lendpeak.com/lp/same-day
lendpeak.com/lp/bad-credit
lendpeak.com/lp/big-ticket
lendpeak.com/lp/mca
lendpeak.com/lp/apply

1 · Add the Google tag to every page <head>

Paste this high in the <head> of all 5 pages. This is our real tag — nothing to change.

<!-- Google tag (gtag.js) -->
<script async src="https://www.googletagmanager.com/gtag/js?id=AW-18139071660"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'AW-18139071660');
</script>

2 · Capture the click ID (gclid) on the form

Add this hidden field inside the pre-qualify <form> and the small script on the page. It reads the gclid Google puts on the ad-click URL and saves it with the lead in GoHighLevel — this is what lets us tie a funded deal back to the exact click later. It survives multi-step forms.

<!-- hidden field inside the <form> -->
<input type="hidden" name="gclid" id="gclid">

<script>
(function(){
  var g = new URLSearchParams(location.search).get('gclid');
  if (g) { try { localStorage.setItem('lp_gclid', g); } catch(e){} }
  var el = document.getElementById('gclid');
  if (el) { try { el.value = g || localStorage.getItem('lp_gclid') || ''; } catch(e){ el.value = g || ''; } }
})();
</script>

3 · Fire the "Lead" event when the form is submitted

Call reportLead(email, phone) right after a lead is saved, passing the email + phone the person entered (Google hashes them in the browser — no raw data leaves). This is the conversion the campaign optimizes on.

<script>
function reportLead(email, phone){
  if (typeof gtag !== 'function') return;
  gtag('set', 'user_data', { email: email || '', phone_number: phone || '' });
  gtag('event', 'conversion', {
    'send_to': 'AW-18139071660/LEAD_LABEL',   // <-- LendPeak sends you this one value
    'value': 1.0,
    'currency': 'USD'
  });
}
// Example, on a successful submit:
// reportLead('jane@hershop.com', '+13105551234');
</script>
One value to receive from LendPeak: the LEAD_LABEL after the slash (the part before it, AW-18139071660, is already correct). LendPeak will send it — paste it in and you're done. Phone format +1XXXXXXXXXX when possible improves matching.

✅ Test (5 min)

  1. Open any of the 5 pages with a test click ID, e.g. lendpeak.com/lp/apply?gclid=TEST123
  2. Submit one test lead (use a fake phone like (424) 555-0142 so no real line rings).
  3. Confirm in GoHighLevel the lead saved with gclid = TEST123.
  4. Tell LendPeak — we confirm the Lead event recorded on our side, then switch the campaign on.
